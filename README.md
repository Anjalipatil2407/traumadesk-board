# 🚑 TraumaDesk Board

## Emergency Dispatch Management System

TraumaDesk Board is a React.js-based emergency dispatch dashboard designed to help emergency responders manage ambulances, prioritize emergency calls, monitor traffic conditions, search patient records, and assign hospitals efficiently.

This project was developed as part of a React JS Case Study.

---

## 📌 Problem Statement

Build a control center for emergency dispatchers to:

* Manage ambulance supplies and crew information
* Prioritize incoming emergency calls
* Search patient medical records
* Rank incidents based on risk level
* Monitor city traffic conditions
* Plan and edit ambulance routes
* Find hospitals with available trauma beds

## ✨ Features

### 🚑 Ambulance Supply List

* Displays ambulance fleet details
* Shows crew information and supply status
* Uses API data

### 🚨 Call Priority Queue

* Sorts emergency calls based on severity
* Critical cases appear first

### 🔍 Patient Record Finder

* Search patients instantly
* Displays patient details fetched from API

### ⚠️ Incident Risk Sorter

* Ranks incidents according to risk level
* Highlights critical emergencies

### 🚦 Traffic Light Status Hub

* Displays traffic conditions at major intersections

### 🗺️ Quickest Route Planner

* Simulates route optimization for ambulances

### ↩️ Route Edit Undo

* Stores route history
* Allows route changes to be reverted

### 🏥 Hospital Bed Finder

* Finds nearest hospitals with available trauma beds
